# wm

Setup project
-----------

* Get Python 2.7
* Install virtualenv
* Run ```virtualenv venv```
* Run ```source venv/bin/activate``` to activate virtual environment
* Run ```pip install -r requirements.txt``` to install project dependencies


Run
---

```python run.py embed -a <algorithm> <img-file> <wm-file>```
```python run.py benchmarks```
```python run.py tests```
